# BudgetFrog
##  Web application for budget tracking
![](https://github.com/MeinLiX/BudgetFrog/tree/master/Resources/DEMO.gif)

## Technologies
Project is created with:
- .NET 6.0 (C#)
- ASP.NET WebAPI (CQRS)
- Entity Framework Core (PostgreSQL provider)
- MediatR
- Node.js (JS)
- Svelte.js
- daisyui (TailwindCSS)
- chart.js
